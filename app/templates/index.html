This is the Chinook Music Database

API Guide

GET All Tracks
curl -X GET http://nns.jkassis.net:5000/api/tracks

GET a Specific Track
curl -X GET http://nns.jkassis.net:5000/api/tracks/1

POST Create a New Track
curl -X POST -H "Content-Type: application/json" -d '{"Name": "New Track", "AlbumId": 12345, "MediaTypeId": 3}' http://nns.jkassis.net:5000/api/tracks

PUT Update a Track
curl -X PUT -H "Content-Type: application/json" -d '{"Name": "Updated Track Name", "AlbumId": 3, "MediaTypeId": 2}' http://nns.jkassis.net:5000/api/tracks/1

DELETE Delete a Track
curl -X DELETE http://nns.jkassis.net:5000/api/tracks/1
